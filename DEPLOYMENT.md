# Spotify ML Prediction App - Deployment Guide

## 🚀 Quick Deploy

### Firebase Hosting (Recommended)
```bash
npm run deploy:firebase
```

### Cloud Run
```bash
npm run deploy:cloudrun
```

### App Engine
```bash
npm run deploy:appengine
```

### Universal Deployment
```bash
./deploy.sh
```
